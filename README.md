___

### Development Setup
```bash
# Install dependencies
yarn install

# Watch for changes to TypeScript, and build on change
gulp watch

# Run the compiled es5 JavaScript code
npm start
```

### Testing
Note: test files should have the suffix: `.spec.ts`
```bash
# Run full test suite
npm test
```
